# Inventory & Supply Chain Management Analysis

This repository contains an end-to-end inventory and supply chain management analysis using a synthetic transactional dataset and interactive Power BI dashboards

**Project Overview**

1.Analyze multi-year order, inventory, and logistics data across regions, categories, warehouses, and suppliers to uncover operational insights.

2.Build KPIs and visuals for inventory turnover, days sales of inventory, warehouse utilization, transportation cost, and backorder performance.

3.Support data-driven decisions in demand planning, inventory control, and logistics optimization.

**Dataset**

The Inventory_SupplyChain_Dataset.csv file includes daily records with fields such as Date, Region, Category, Supplier, Warehouse, Order Status, Units Sold, Inventory Level, Transportation Cost, Order Accuracy, Lead Time (Days), Backorder flag, COGS, Average Inventory, and Warehouse Capacity.


**Dashboards & KPIs**

The Power BI report (documented in the PDF) summarizes key metrics including:

1.Warehouse utilization, days sales of inventory, and inventory turnover ratio by region and category.

2.Transportation cost by region/category and units sold over time.

3.Lead time by product category, backorders by order status, and inventory levels by region
